Sobre el proyecto

La aplicación de mangas permite gestionar de manera eficiente una colección de mangas mediante operaciones CRUD (Crear, Leer, Actualizar, Eliminar). Desarrollada con Node.js y Express en el backend, y utilizando SQLite como base de datos, proporciona una experiencia fluida y rápida para el usuario. El frontend está construido con Angular, ofreciendo una interfaz intuitiva que facilita agregar, editar, consultar y eliminar mangas de forma sencilla y efectiva.
